Exchange.io

Currency exchange rates
and conversion API

Usage:

Get the latest currency exchange rates in JSON format.

http://192.168.0.171:8000/api/latest/ 

                        
                    

            

            

            

                
Get rates back to 90 days ago.

                    

                        
                                             
                    

                    

                        

                            http://192.168.0.171:8000/api/2018-03-14
                        

                        
                        
                    

            

            

            

                
Rates are quoted against the EURO by default. To quote against a different currency use the
                    base uri parameter in your request.

                    

                            
                                                 
                        

                        

                            

                                http://192.168.0.171:8000/api/latest?base=USD&symbols=EUR,GBP,JPY
                            

                            
                        
                        

            

            

            

                
Request specific exchange rates by setting the
                    symbols parameter.

               
                    

                            
                                                 
                        

                        

                            

                                http://192.168.0.171:8000/api/latest?symbols=USD,GBP
