For clients to authenticate, the token key should be included in the Authorization HTTP header. The key should be prefixed by the string literal "Token", with whitespace separating the two strings.

For example:

Authorization: Token YOUR_TOKEN_KEY

And this header should be included for all the requests (below), otherwise you will be throttled after sending 10 requests without the Authorization header and Token.


⚠ Requests bellow are not authenticated.

Code Samples:

            

                
Python code sample to fetch the latest currencies exchange rates:

                
                
import requests
headers = {"Authorization": "Token YOUR_TOKEN_KEY"}
response = requests.get('http://www.exchangeio.com/api/latest/', headers=headers)
respJSON = response.json()
print(respJSON)

                
            

            

                
Javascript code sample to fetch the latest currencies exchange rates:

                
                
fetch('http://www.exchangeio.com/api/latest/', {
    method: "GET",
    headers: {
        "Authorization": "Token YOUR_TOKEN_KEY"
    }
})
.then(response => response.json())
.then(console.log)
.catch(console.error);
